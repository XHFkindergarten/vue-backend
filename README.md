# A personal backend project 
by XHFkindergarten@gmail.com



## 用户模块

``` 
登录
# 密码加密
# token验证
# 保留登录状态

注册
# 邮箱验证

修改用户头像

路由
# 无线路由嵌套
# 根据用户权限动态挂载路由
# 自动生成路由侧边栏
# 支持管理员动态添加路由权限（未完成）
```

## 正式功能

```
文章

依赖simpleMde组件
# Markdown语法实时预览
# 文章分组
# 实时预览
# 控制发布状态
# 评论回复模块
# 阅读页面

doc/wpf格式实时朗读

依赖百度AI语音引擎
# 允许导入4M以内的文件(过大的文件会timeout)
# 提供音量、音色（男女）等选项

动态发布功能

# 时间线发布形式
# 支持九图

```

``` bash
# build for production with minification
npm run build
```
